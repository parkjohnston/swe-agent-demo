# TypeScript Project Structure

This document outlines the standard file structure for a TypeScript project.

## Project Structure

```
/your-project-root
|-- /src
|   |-- /components
|   |-- /services
|   |-- /utils
|   |-- index.ts
|-- /dist
|-- /tests
|-- package.json
|-- tsconfig.json
|-- README.md
```

### /src
- **components/**: Contains all the React components or any UI components.
- **services/**: Contains service classes or functions for API calls and business logic.
- **utils/**: Contains utility functions and helpers.
- **index.ts**: The entry point of the application.

### /dist
- Compiled output files.

### /tests
- Contains all test files.

### package.json
- Contains project metadata and dependencies.

### tsconfig.json
- TypeScript configuration file.

### README.md
- Project documentation.
